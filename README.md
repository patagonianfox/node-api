# WebServer + RestServer

Al inicio ```npm install``` para reconstruir los módulos de Node.

## Levantar aplicación
```npm start```
or
```nodemon app```

## DB
MongoDB Cloud Server

## API REST
### Endpoints 
#### Usuarios
GET /api/usuarios
GET /api/usuarios?offset=0&limit=5
#### Login
POST /api/auth/login
### JWT
Header: x-token [Deprecado]
Header: Authorization Bearer Token
